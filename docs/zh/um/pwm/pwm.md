# PWM

PWM 设备支持使用 PWM 信号输出，同时使用 ICU (输入捕获组) 功能。

支持的硬件：

| 制造商    | Protocols                          |
| ------ | ---------------------------------- |
| ECUBUS | PWM, ICU (正在进行) |

![pwm](image.png)

## PWM 互动

通过 `Network` 窗口打开 PWM IA。 IA 能力：

- 设置PWM 任务周期 (支持多个设备)
- Quikc 预设：
  - 0%
  - 25%
  - 50%
  - 75%
  - 100%
