# 图表（Graph）

图表信号功能可将信号之间的数据流以图形方式可视化。该功能有助于理解信号间的关系，并用于调试复杂的信号交互。
![图表示意](image.png)

* **Line（折线）**
* **Gauge（仪表）**
* **Data（数据）**
