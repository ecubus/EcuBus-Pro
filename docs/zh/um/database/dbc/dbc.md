# CAN DBC

EcuBus-Pro支持解析大多数CAN DBC文件。若遇到无法正确解析的DBC文件，请在我们的[Github Issues](https://github.com/ecubus/EcuBus-Pro/issues)页面提交问题报告。

该应用程序为消息和信号提供了高效的搜索界面。

> [!NOTE]
> 目前，EcuBus-Pro仅支持查看DBC文件，不支持编辑功能。使用前请确保您的DBC文件格式正确无误。
