# PWM

PWM 设备支持输出 PWM 信号，同时具备 ICU（输入捕获单元）功能。

支持的硬件：

| 制造商    | 协议           |
| ------ | ------------ |
| ECUBUS | PWM、ICU（进行中） |

![pwm](../../../media/um/pwm/image.png)

## PWM 交互

通过 `Network` 窗口打开 PWM IA。 IA 功能： IA 功能：

- 设置 PWM 占空比（支持多个设备）
- 快速预设：
  - 0%
  - 25%
  - 50%
  - 75%
  - 100%
