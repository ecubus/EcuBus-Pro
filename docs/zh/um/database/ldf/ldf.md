# LIN LDF

LDF 解析器支持以下LIN 规格版本：

- 2.2
- 2.1
- 2.0

## 选择一个 LDF 文件

![替代文本](../../../../media/um/database/ldf/image-4.png)

## 保存数据库

创建新数据库时，您必须指定一个唯一的名称并保存。
![替代文本](../../../../media/um/database/ldf/image-5.png)

## LDF 编辑器

您可以通过不同标签编辑LDF数据库。 每一节都严格检查错误，以确保数据的完整性。

### A. 概况

此选项卡显示一般数据库信息和错误通知。 保存或使用数据库前必须解决所有错误。
![替代文本](../../../../media/um/database/ldf/image-6.png)

### Nodes

配置此部分的从属节点。 严格的错误检查是为了验证节点配置。
![替代文本](../../../../media/um/database/ldf/image-7.png)
![替代文本](../../../../media/um/database/ldf/image-8.png)

### 信号

管理此选项卡中的信号信息。 将突出强调错误信号。 所有信号配置都经过严格验证。
![替代文本](../../../../media/um/database/ldf/image-9.png)
![替代文本](../../../../media/um/database/ldf/image-10.png)

### Frames

本节只显示无条件框架。 其他框架类型可以在日程表中配置。 将突出强调框架错误。 严格的错误检查确保框架的完整性。
![替代文本](../../../../media/um/database/ldf/image-11.png)
![替代文本](../../../../media/um/database/ldf/image-12.png)

### 日程安排

配置调度表并在此节中添加帧。 所有计划配置都经过严格验证。
![替代文本](../../../../media/um/database/ldf/image-13.png)
![替代文本](../../../../media/um/database/ldf/image-14.png)

### 编码

在此定义编码信息。 信号编码可以在配置后在信号选项卡中编辑。 所有编码定义都进行严格验证。
![替代文本](../../../../media/um/database/ldf/image-15.png)
![替代文本](../../../../media/um/database/ldf/image-16.png)

### LDF

查看实时的 LDF 文件内容。 一旦解决了所有错误，您可以从这里复制LDF内容。
![替代文本](../../../../media/um/database/ldf/image-17.png)
