# 诊断仪在线

诊断仪在线（服务ID 0x3E）是一种诊断服务，用于向ECU表明诊断客户端（诊断仪）仍然在线，诊断会话应保持激活状态。

诊断仪在线的主要目的是：

- 通过定期发送消息来保持非默认诊断会话处于激活状态
- 防止诊断会话超时（S3超时）
- 维持诊断仪与ECU之间的通信

> [!NOTE]
> 诊断仪在线目前仅支持CAN。

## 配置

在EcuBus-Pro中，诊断仪在线可在`UDS Tester`窗口中配置：

![诊断仪在线配置](./images/tester_present.png)

### S3时间[M]

S3时间指S3客户端超时时间，即诊断仪在线消息之间的最长时间间隔。

### 诊断仪在线地址[M]

诊断仪在线消息的地址，可设置为物理地址或功能地址。

### 来自特殊服务[O]

诊断仪在线的默认消息为`0x3E 0x00`，如需使用特殊服务，可从服务列表中选择一个。
