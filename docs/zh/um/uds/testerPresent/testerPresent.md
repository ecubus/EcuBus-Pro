# 测试仪在线

测试仪在线（服务 ID 0x3E）是一种诊断服务，用于向 ECU 指示诊断客户端（测试仪）仍然在线且诊断会话应保持活动状态。

测试仪在线的主要目的是：

- 通过定期发送消息来保持非默认诊断会话处于活动状态
- 防止诊断会话超时（S3 超时）
- 维持测试仪与 ECU 之间的通信

> [!NOTE]
> 测试仪在线目前仅支持 CAN。

## 配置

在 EcuBus-Pro 中，可以在 `UDS 测试仪` 窗口中配置测试仪在线：

![测试仪在线配置](../../../../media/um/uds/testerPresent/images/tester_present.png)

### S3 时间 [M]

S3 时间表示 S3 客户端超时，它是测试仪在线消息之间的最长时间。

### 测试仪在线地址 [M]

测试仪在线消息的地址，可以设置为物理地址或功能地址。

### 来自特殊服务 [O]

测试仪在线的默认消息是 `0x3E 0x00`，如果您想使用特殊服务，可以从服务列表中选择一个。
