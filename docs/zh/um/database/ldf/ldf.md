# LIN LDF

LDF 解析器支持以下 LIN 规范版本：

- 2.2
- 2.1
- 2.0

## 选择 LDF 文件

![alt text](../../../../media/um/database/ldf/image-4.png)

## 保存数据库

创建新数据库时，您必须为其分配唯一名称并保存。
![alt text](../../../../media/um/database/ldf/image-5.png)

## LDF 编辑器

您可以通过不同的选项卡编辑 LDF 数据库。 每个部分都实施了严格的错误检查，以确保数据完整性。

### 常规

此选项卡显示常规数据库信息和错误通知。 在保存或使用数据库之前，必须解决所有错误。
![alt text](../../../../media/um/database/ldf/image-6.png)

### 节点

在此部分配置从节点属性。 实施了严格的错误检查以验证节点配置。
![alt text](../../../../media/um/database/ldf/image-7.png)
![alt text](../../../../media/um/database/ldf/image-8.png)

### 信号

在此选项卡中管理信号信息。 错误的信号将被高亮显示。 所有信号配置都经过严格验证。
![alt text](../../../../media/um/database/ldf/image-9.png)
![alt text](../../../../media/um/database/ldf/image-10.png)

### 帧

此部分仅显示无条件帧。 其他帧类型可以在调度表中配置。 错误的帧将被高亮显示。 严格的错误检查确保帧完整性。
![alt text](../../../../media/um/database/ldf/image-11.png)
![alt text](../../../../media/um/database/ldf/image-12.png)

### 调度

在此部分配置调度表并向其中添加帧。 所有调度配置都经过严格验证。
![alt text](../../../../media/um/database/ldf/image-13.png)
![alt text](../../../../media/um/database/ldf/image-14.png)

### 编码

在此处定义编码信息。 配置后，可以在信号选项卡中编辑信号编码。 对所有编码定义执行严格验证。
![alt text](../../../../media/um/database/ldf/image-15.png)
![alt text](../../../../media/um/database/ldf/image-16.png)

### LDF

查看实时 LDF 文件内容。 解决所有错误后，您可以从此处复制 LDF 内容。
![alt text](../../../../media/um/database/ldf/image-17.png)
