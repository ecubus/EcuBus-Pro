## Prerequisites

- Node.js (v22)
- npm
- Node-gyp

## Installation Steps

```bash
git clone https://github.com/ecubus/EcuBus-Pro.git
```

```bash
cd EcuBus-Pro
```

```bash
npm install
```

## Build Steps

### Build Documentation

```bash
npm run docs:build
```

### Build Native Module

```bash
npm run native
```

### Build Worker

```bash
npm run worker
```

### Build API

```bash
npm run api
```

### Build CLI (Windows)

```bash
npm run cli:build:win
```

### Run Development

```bash
npm run dev
```

### Build Application (Windows)

```bash
npm run build:win
```
