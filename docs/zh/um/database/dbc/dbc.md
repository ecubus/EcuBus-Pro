# CAN DBC

EcuBus-Pro 支持解析大多数 CAN DBC 文件。 EcuBus-Pro 支持解析大多数 CAN DBC 文件。 如果您遇到任何无法正确解析的 DBC 文件，请在我们的 [Github Issues](https://github.com/ecubus/EcuBus-Pro/issues) 页面上报告问题。

该应用程序为报文和信号提供了高效的搜索界面。

> [!NOTE]
> 目前，EcuBus-Pro 仅支持查看 DBC 文件，不支持编辑。 请确保您的 DBC 文件在使用前格式正确。 请确保您的 DBC 文件在使用前格式正确。
