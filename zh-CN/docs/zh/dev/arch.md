# 结构

该项目的架构基于`Electron`框架。 它是构建带有JavaScript、 HTML和 CSS 的跨平台桌面应用程序的流行框架。
关于 `Electron` 框架的更多信息可以找到 [electronjs](https://www.electronjs.org/)。

## 技术应用

- `Electron` - 构建桌面应用程序的核心框架。
- `Vue` - 构建用户界面的前端框架。
- `TypeScript` - 构建应用程序的编程语言。
- `Vite` - 项目的构建工具。
- `Webpack` - 项目的构建工具。
- `Electron-builder` - 用于包装应用程序的构建工具。
- `ElementPlus` - 用于构建用户界面的 UI 库。
- `Vite-Test` - 项目的测试工具。
- `node-addon-api` - 构建本机节点模块的API。
- `node-gyp` - 本机节点模块的构建工具。
- `swig` - 项目的模板引擎。

我们使用 [Electron-Vite](https://electron-vite.org/) 将这些堆栈合并在一起，这是构建`Electron`应用程序与 `Vite`的模板。
