## Prerequisites

- Node.js (v22)
- npm
- Node-gyp

## Installation Steps

```bash
git clone https://github.com/ecubus/EcuBus-Pro.git
```

```bash
cd EcuBus-Pro
```

```bash
npm install
```

`build native module`

```bash
npm run native
```

`run`

```bash
npm run dev
```

`build`

```bash
npm run build:win
```
