## 客户端与服务器端JavaScript

当JavaScript在网页上被使用时，它是在网页浏览器中执行的，作为客户端语言。

JavaScript 可以在 web 浏览器和服务器上运行。 受欢迎的 JavaScript 服务器端环境是 Node.js。 不同于客户端JavaScript，服务器端JavaScript在服务器上执行并允许您访问数据库、文件系统等。

Node.js不是Python、 Java 或 C/C++等编程语言。 Node.js是一个运行时间，类似于Java 虚拟机将JavaScript代码转换为机器代码。 全世界数以千计的开发者广泛使用它来开发 I/O 密集的网络应用程序，如视频流站点。 单页应用程序和其他网页应用程序。

- [Node.js 教程 1](https://www.w3schools.com/nodejs/)
- [Node.js 教程 2](https://www.tutorialspoint.com/nodejs/index.htm)
