## 必备条件

- Node.js (v22)
- npm
- 节点健身房

## 安装步骤

```bash
git 克隆https://github.com/ecubus/EcuadBus-Pro.git
```

```bash
cd EcuBus-Pro
```

```bash
npm install
```

## 构建步骤

### 构建文档

```bash
npm 运行文档：build...
```

### 构建本机模块

```bash
npm 运行本机
```

### 构建工人

```bash
npm 运行工人
```

### 生成 API

```bash
npm 运行 api
```

### 构建CLI (Windows)

```bash
npm 运行客户端:build:win
```

### 运行开发

```bash
npm 运行dev
```

### 构建应用程序 (Windows)

```bash
npm 运行构建:win
```
